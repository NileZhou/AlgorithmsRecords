本菜逐渐感受到掌握一些基础的算法代码片段的重要性，看到在做难题时很多高手信手拈来快速写代码，仿佛不需要进行思考。
我猜他们是做到了两点: 
- 快速拆解题目，想出可能的解法
- 将解法转化为自己熟悉的代码片段,快速实现

拆解题目这一步我相信没有捷径，唯有勤加苦练
本菜想做的是在不断学习的过程中，解决第二个问题

算法之路漫漫，唯有上下而求索
期待小伙伴加入
